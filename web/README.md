# OPM Federal Workforce Visualization Catalog

A searchable, filterable catalog of 22 OPM federal workforce data visualizations across 6 analytics categories.

## Features

- **Real-time search** - Search by title or example questions
- **Category filtering** - Filter by workforce category (6 categories)
- **Variable filtering** - Click variable pills to filter dashboards by specific data fields (Department, Agency, Occupational Series, etc.)
  - **ANY mode**: Show dashboards with any selected variable
  - **ALL mode**: Show dashboards with all selected variables
- **Responsive design** - Works on desktop, tablet, and mobile
- **Direct links** - One-click access to OPM pages
- **No dependencies** - Pure HTML/CSS/JS, no build process needed

## Deploy to Netlify

### Option 1: Drag & Drop
1. Go to [Netlify Drop](https://app.netlify.com/drop)
2. Drag the `index.html` file onto the page
3. Done! Your site is live

### Option 2: GitHub + Netlify
1. Create a new GitHub repository
2. Push `index.html` to the repository
3. Go to [Netlify](https://app.netlify.com/)
4. Click "New site from Git"
5. Connect your GitHub repository
6. Deploy!

### Option 3: Netlify CLI
```bash
# Install Netlify CLI
npm install -g netlify-cli

# Deploy
netlify deploy --prod --dir .
```

## What's Included
